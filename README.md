# Hoosier Daddy HVAC - Static Site

This repository contains the static HTML files for Hoosier Daddy HVAC.

## Deployment
The site is deployed automatically to Netlify from the `pages` directory.

### Build Configuration
- **Publish Directory:** `pages`
- **Build Command:** (None/Static)

## Project Structure
- `/pages`: Contains all live site content, including HTML, CSS, JS, and images.
- `netlify.toml`: Configuration for Netlify deployment.
